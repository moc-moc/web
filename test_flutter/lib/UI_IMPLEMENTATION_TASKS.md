# UI実装タスクリスト

このファイルは、プロジェクトのUI実装タスクを段階的にリスト化したものです。
完了したタスクにチェックを入れて進捗を管理してください。

**重要な方針**:
- すべてダミーデータで動作させる（バックエンド機能は未実装）
- 画面遷移は自由にテスト可能にする
- デザインシステムを先に構築してから画面を実装
- カラー・コンポーネントは中央管理（theme.dart、widgets/）
- 統一感のあるデザイン（角丸、余白、ボタンスタイルを統一）
- **画面作成時は必ずルーティングに登録してUIで確認可能にする**
- **関連する画面から遷移できるようにする（不明な場合は設定画面から）**
